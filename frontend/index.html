UNCCD GeoGLI Chatbot

Welcome to the Global Land Indicator chatbot. Ask questions about land degradation, desertification, and sustainable land management.

The green chat widget in the bottom-right corner is ready to help you explore land indicator data and insights.

Try asking:

  • "What are the main land degradation indicators?"
  • "How is vegetation productivity measured?"
  • "Tell me about soil organic carbon trends"
  • "What is the percentage breakdown of land use in Kenya for agriculture, urban areas, and protected areas?"
  • "What was the total area of land degradation in Kenya between 2001 and 2015?"
  • "wildfires in Saudi Arabia"
  • "show Saudi stressors"
  • We implemented four main query intents: ask.country – country-level information (e.g., Saudi Arabia wildfires). ask.region – regional information. ask.commit_country – country commitments. ask.commit_region – regional commitments. Post-processing logic ensures: If BM25 finds hits → directly return cards (with title, text, citation, and images). If no hits and dense retrieval is disabled → return fallback from the LLM. Streamed responses are supported via SSE for integration with the frontend. I mainly worked on data ingestion (snapshots + citations), BM25 setup, fixing query endpoints (GET/POST), and enabling image rendering in the chat cards.
